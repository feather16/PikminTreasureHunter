# PikminTreasureHunter
ピクミンをモチーフにした2Dゲームです。

制限時間内にお宝を全て集めればクリアとなります。

全部で16のコースがあります。

WASDで移動、Enterでアクション。

[DXライブラリ](https://dxlib.xsrv.jp/)を用いて開発しました。
プログラミングを始めて間もない頃に書いたコードなので、読みにくいかもしれません。

# 実行
`run.bat`
