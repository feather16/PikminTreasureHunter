# PikminTreasureHunter
ピクミンをモチーフにした2Dゲームです。

制限時間内にお宝を全て集めればクリアとなります。

全部で16のコースがあります。

WASDで移動、Enterでアクション。

[DXライブラリ](https://dxlib.xsrv.jp/)を用いて開発しました。
プログラミングに慣れていない頃に書いたコードなので、読みにくいかもしれません。

画像は自分で描き、SEはフリー素材を使用しています。

# 動作環境
Windows(10で動作することを確認済み)

# 実行
`run.bat`
